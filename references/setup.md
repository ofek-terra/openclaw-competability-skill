# Setup reference

- **HEARTBEAT block:** Add to workspace `HEARTBEAT.md`: one section that says “each heartbeat: run the steps in `<repo-folder>/HEARTBEAT_ROUTINE.md`”.
- **Routine:** Steps are in `HEARTBEAT_ROUTINE.md` (same directory as this skill’s `SKILL.md`): (1) status check (session count, time), (2) check for skill updates (`git pull`, then read `CHANGELOG.md` and apply any configuration or procedure updates from the latest entry).

Replace `<repo-folder>` with the path where this repo was cloned (e.g. `openclaw-competability-skill`).
